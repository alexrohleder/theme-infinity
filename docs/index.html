Build


Here are described the proccess to rebuild the theme from the raw source files, the build are made throght grunt. If you are not familiar with that you can read the comments on the /gruntfile.js they describe all the tasks that will be run when you execute the grunt.

For start build you only need to run 2 commands, the first are with npm for installing the theme dev packages

npm install

And the second to start the building. Note that the first command is runned once, after that when you want to rebuild only execute this command:

grunt

Sending E-Mails

This theme includes a fully function way to send e-mail directly from the form without need of a backend, only javascript!

For this you will need an account on Mandrillapp they free plans enable you to send 2.000 e-mails each month. After the account creating you will need to get your API key, for this go to settings on your dashboard and create a new one.

And you are ready to use the full functional form, see above the javascript code included on this theme:

            
function sendEmailWithMandril(data) {
    $.post('https://mandrillapp.com/api/1.0/messages/send.json', {
        key: '**you mandrillapp api key**',
        message: {
            autotext: 'true',
            subject: data.name + ' got in touch via portfolio',
            from_email: data.email,
            html: '<p>' + data.name + ' <b>' + data.email + '</b></p><p>' + data.message + '</p>',
            to: [{
                email: '**the email you want to receive the emails**',
                name : '**your name**',
                type : 'to'
            }]
        }
    })
}
